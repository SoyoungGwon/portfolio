SKILL

1 HTML5

새로운 웹 표준 문서 규격인 HTML5 규격으로 개발된 웹사이트는 각기 다른 웹브라우저나 운영체제를 사용하더라도 동일하게 웹페이지를 표시할 수 있으며, 액티브엑스와 같은 플러그인의 설치 없이도 자체적으로 음악이나 동영상의 재생이 가능하도록 설계, 제작할 수 있습니다.

2 CSS3, SASS

CSS3는 웹문서의 전반적인 스타일을 미리 저장해 둔 스타일 시트인 CSS의 최신버전입니다. 그러나 CSS의 유지보수 및 효율성 이슈로 SASS가 등장하게됩니다. 여기서 SASS란 보다 간결하고 격식을 갖춘 CSS문법을 제공하는 CSS의 전처리기이며, 스타일시트를 쉽게 관리 할 수 있는 다양한 기능을 구현할 수 있습니다.

3 JS, JQuery

JavaScript란 동적으로 컨텐츠를 바꾸고, 멀티미디어를 다루고, 움직이는 이미지등 웹 페이지를 꾸며주도록 하는 프로그래밍 언어입니다. JQuery 즉 자바스크립트 라이브러리를 사용하면 JS를 좀 더 쉽고 간단하게 사용할 수 있습니다.

4 Github

git은 프로그램 등의 소스코드 관리를 위한 분산 버전 관리 시스템입니다. github는 git을 호스팅해주는 웹 서비스이며, git 저장소 서버를 대신 유지 및 관리해주기 때문에 보다 효과적으로 프로젝트를 관리 할 수 있습니다.
Github 주소 : https://github.com/SoyoungGwon/portfolio
